# Recomendaciones

## Instalación VPS y adjuntar dominio

Para realizar todo lo del siguiente video se requiere usar lo siguiente para evitar problemas.

> OVH Cloud para VPS: [https://www.ovhcloud.com/es/](https://www.ovhcloud.com/es/)
>
> Hostinger para dominio: [https://www.hostinger.com/mx/dominios](https://www.hostinger.com/mx/dominios)\
> Cloudflare para manejar el dominio y seguridad: [https://dash.cloudflare.com/](https://dash.cloudflare.com/)
>
> PgAdmin4 para manejar la base de datos: [https://www.pgadmin.org/download/](https://www.pgadmin.org/download/)

Primero se debe adquirir un servicio dentro de la plataforma de OVH Cloud para poder continuar
